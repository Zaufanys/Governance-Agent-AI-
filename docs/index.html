How It Works

1

Agent Runs

Your agent executes a task with tools and context.

2

Ingest Trace

POST trace data to /api/ingest with your API key.

3

Score Risk

Engine scores the trace against 8 governance rules in real-time.

4

Review & Decide

Humans approve, reject, or escalate high-risk runs.

5

Audit Export

Full audit bundle with traces, scores, and decisions.

Governance Rules

Rule Category Weight Purpose
Direct Prompt Injection Security 35 Detects override attempts in user input
Indirect Injection Security 30 Detects poisoned retrieved content
Write Tool Without Approval Control 30 Flags state-changing actions without review
Sensitive Data Data 25 Detects PII, credentials, or confidential info
Customer-Facing Without Review Control 20 Requires approval for user-visible output
Low Groundedness Quality 15 Flags outputs lacking evidence support (< 0.75)
Excessive Tool Calls Quality 10 Flags runs with > 5 tool invocations
High Latency Quality 5 Flags runs taking > 2000 ms
